Octoliner Geoflow's documentation!
==================================

Welcome to the Octoliner Geoflow documentation.

Octoliner GeoFlow is a module that can be used to Parse Filter and write GeoJSON file in a fast way. using C++ and has an interface in C++, Python, and Javascript

This documentation is aimed for developper that want to contribute to the project. or user than want to exploit the potential of this library for their usecase.
