Doubly Connected Edge List
==========================

Doubly Connected Edge List(DCEL) is a data structure that topologicaly represent GeoJSON Data before being treated.
DCEL has a subtility compared to raw GeoJSON data is that Face edges that touch each other share the same DCEL Edge.

The DCEL namespace can be use to convert GeoJSON data to DCEL exploitable data Vertex/Helf_Edge/Face
